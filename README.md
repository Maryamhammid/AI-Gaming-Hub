| Game            | AI Used                     | File           |

##  Features

-  Smart AI opponents for each game
-  Modern UI with animations and smooth transitions
-  Game status indicators: player turns, win/draw states
-  Highscore tracking (JSON)

##  Tech Stack

- **Language**: Python
- **Game Engine**: Pygame
- **AI Techniques**: Minimax, Expectimax, Heuristics
- **Storage**: JSON
